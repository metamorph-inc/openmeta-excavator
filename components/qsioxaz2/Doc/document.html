Class Description

Class Usage

The panel is not used as a component itself, it is used to create panel components, custom components, and custom assemblies that in turn must become components within the library. No specific restrictions are placed on a panel other than you can only remove, not add, material to create a component. If material needs to be added, then it becomes a custom assembly.

Class Connections

The panel has no connections identified due to the non-specific nature of the component.
